# TypeScript Study

- TypeScript를 사용하여 효과적으로 타입을 사용하는 방법

### 1. Basic

- 타입 스크립트의 기본 타입
- 타입과 인터페이스 이름 짓기
- 객체 형태처럼 내부에서 타입을 지정해 줄 수 있는 타입
- 타입 추론 캐스팅, 구체화된 값 지정
- 큐플 Array 안에서도 구체화된 값 지정
